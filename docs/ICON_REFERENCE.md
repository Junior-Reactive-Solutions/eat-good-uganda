# Icon Reference & Visual Overview

This document provides a visual reference for all hero icons in the Eat Good Uganda icon system. Each icon is documented with its design intent, specifications, and usage guidelines.

---

## Payment Method Icons

### 1. MTN Mobile Money (MoMo)

**File:** `icon-payment-momo.svg`  
**Category:** Payment  
**Size:** 24×24px (2px stroke)  
**Color Scheme:** #333333 outline + #FF6B35 accent

#### Design Intent

MTN Mobile Money (MoMo) is Uganda's most-used mobile payment method. This icon represents quick, easy digital payments through a phone-based system.

#### Visual Design

```
┌─────────────────────┐
│    Outer circle      │  ← 22px diameter, 2px stroke
│  ┌───────────────┐   │
│  │   PHONE       │   │  ← Phone body, 6×10px
│  │    FRAME      │   │
│  │      ₦        │   │  ← Money symbol (₦) in warm orange
│  │   (Orange)    │   │
│  └───────────────┘   │
│     ●(accent dot)    │  ← Small warm accent on frame
└─────────────────────┘
```

#### Key Features

- Circle frame creates friendly, safe feeling
- Phone silhouette instantly recognizable
- Money symbol (#FF6B35 warm accent) highlights payment purpose
- Small accent dot adds character
- All corners rounded, no sharp angles

#### Usage

Use on payment selection screens, checkout flows, and payment method indicators.

#### Design Specifications

- Circle outer radius: 11px
- Circle inner stroke: 2px
- Phone dimensions: 6×10px
- Phone position: centered
- Corner radius: 1px (phone corners)
- Accent color placement: Money symbol and accent dot

## Design Language Summary

### Warm Aesthetic Characteristics

All icons demonstrate these design principles:

1. **Rounded Corners**: Minimum 1px radius everywhere
2. **Curved Lines**: No sharp angles or rigid geometry
3. **Consistent Stroke**: 2px throughout (at 24px size)
4. **Warm Accent Color**: #FF6B35 used on 1–2 key elements per icon
5. **Hand-Crafted Feel**: Slight asymmetry and organic quality
6. **Grid Alignment**: Perfect 24px grid positioning
7. **Cultural Relevance**: Icons reflect Uganda's payment/delivery ecosystem
8. **Clear at Small Sizes**: Simplified shapes remain recognizable at 24px

### Color Palette Reference

| Element         | Color       | Hex     | Usage                         |
| --------------- | ----------- | ------- | ----------------------------- |
| **Outline**     | Dark Gray   | #333333 | Primary icon strokes          |
| **Warm Accent** | Warm Orange | #FF6B35 | Key highlights (1–2 per icon) |
| **Disabled**    | Light Gray  | #CCCCCC | Unavailable states            |
| **Focus Ring**  | Focus Blue  | #2196F3 | Keyboard focus indicator      |

### Sizing & Scaling

All icons designed at 24px with 2px stroke:

- **24px**: 2px stroke (controls, small UI)
- **32px**: 2.67px stroke (standard display)
- **48px**: 4px stroke (large buttons, hero sections)
- **64px**: 5.33px stroke (marketing, high-prominence)
